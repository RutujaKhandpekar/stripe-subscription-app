# stripe-subscription-app
This will create the stripe subscrition for a customer and the card payment will be done.

# .env.local
Provide your publishable and secret key. Also, productID and customerID are hard-coded for now.

# npm install
This will install the dependencies.
This should create the node-modules folder in the same repository.

# npm run dev
Runs the app in the development mode.
Open http://localhost:3000 to view it in your browser.
